README for Quadsolver for CS 4900 - By Jared Teller and Joe Manto

-------------Compile and Run Instructions-----------------
To run the core program use "make quad" which will run the
executable ./quadsolver and start prompting for user input.
To create the executable run "make".
__________________________________________________________

-------------------------Version Control------------------
Git was used for version control of the program and allowed
for effective pair programming.
__________________________________________________________

-------------------------Standards------------------------
We decided to use the C11 standard for gcc. 
The program will be run with Make instructions like -Wall 
-pedantic.
